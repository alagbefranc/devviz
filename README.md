# 🤖 DevViz: AI-Powered Development Visualization Platform

> Unified development platform combining Git workflows, API design, and database schema visualization with intelligent AI assistance

## 🚀 Quick Start

```bash
# 1. Clone repository
git clone https://github.com/yourusername/devviz-platform.git
cd devviz-platform

# 2. Setup Supabase (automated with MCP)
# 3. Setup frontend (Angular + ng-diagram)
# 4. Setup AI API (Vercel AI SDK)
# 5. Start development servers
```

## 📋 Project Structure

```
devviz-platform/
├── frontend/           # Angular + ng-diagram application
├── ai-api/            # Vercel AI functions
├── supabase/          # Database migrations and setup
└── docs/              # Documentation
```

## 🛠️ Technology Stack

- **Frontend**: Angular 18+ + ng-diagram + TypeScript
- **AI Backend**: Vercel AI SDK + OpenAI/Claude
- **Database**: Supabase (PostgreSQL + Realtime)
- **Deployment**: Vercel + Supabase

## 🎯 Features

- 🌿 **Git Workflow Designer**: Visual branch management
- 🔌 **API Designer**: RESTful API design with OpenAPI generation
- 🗄️ **Database Schema Designer**: Visual database modeling
- 🤖 **DevBot AI Assistant**: Context-aware AI assistance
- 👥 **Real-time Collaboration**: Live team editing

## 🚀 Development

### Prerequisites
- Node.js 18+
- Angular CLI 19+
- Supabase CLI
- OpenAI API key

### Setup Commands
```bash
# Install dependencies
npm install

# Start development servers
npm run dev:frontend    # Angular dev server
npm run dev:ai-api     # Vercel dev server
npm run dev:supabase   # Supabase local development
```
